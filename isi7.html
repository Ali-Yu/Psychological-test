/ 失眠与心理健康的关联

大型中国青少年研究（N = 570,295）发现 ISI 与 PHQ-9 的相关系数 r = 0.72，提示失眠与抑郁高度共病（Wang et al., 2025）。失眠既可以是焦虑、抑郁的症状，也可以是它们的诱因——睡眠剥夺会显著提升负面情绪的反应性。如果你同时注意到情绪低落或焦虑感明显，建议在寻求专业帮助时一并提及。

/ 循证干预方法

  • 认知行为治疗失眠（CBT-I）：现有最高证据等级的失眠干预，包括睡眠限制、刺激控制、认知重构三大核心成分，长期效果优于安眠药物（Trauer et al., 2015 Annals of Internal Medicine）
  • 刺激控制：只在困意强烈时上床；床只用于睡眠，不用于刷手机、工作；20分钟睡不着就起床，直到有睡意再回来
  • 固定起床时间：无论前一晚睡了多久，固定同一时间起床——这是重建昼夜节律最有效的单一行为
  • 减少"努力入睡"：越努力入睡，大脑越警觉。尝试"矛盾意向"——刻意保持清醒（但放松躺着），反而更容易入睡
  • 减少补偿行为：避免白天过度补觉、提早上床或延迟起床——这些行为会维持失眠的恶性循环

/ 参考常模与研究背景

ISI 由睡眠研究学者 Charles Morin 于 1993/1996 年开发，是全球最广泛使用的失眠自评工具之一，已被翻译为中文、意大利语、韩语、葡萄牙语、阿拉伯语、德语等十余种语言并完成心理测量验证。量表包含7个条目，分别评估失眠严重程度（入睡困难、维持睡眠、早醒）、睡眠满意度，以及失眠对日间功能的影响、他人的可察觉性和主观困扰程度，采用0–4五级计分，总分0–28分。

信效度：可靠性泛化元分析（33项研究，N = 29,688，α范围 0.65–0.92）显示加权平均 Cronbach's α = 0.82（Cerri et al., 2023）；原版两项大样本研究（社区样本 N = 959，临床样本 N = 245）内部一致性均为 α = 0.90–0.91（Morin et al., 2011）。与匹兹堡睡眠质量指数（PSQI）相关 r = 0.80，效标效度良好。

分段临界值：以半结构化临床访谈为金标准，社区样本中临界值10分具有最优判别性能（灵敏度 86.1%，特异度 87.7%）；临床样本推荐临界值15分用于识别中重度临床失眠（Morin et al., 2011）。治疗反应方面，总分变化 ≥8.4 分对应"中度改善"，≥9 分对应"显著改善"。

人群分布与性别差异：中国青少年大规模研究（N = 570,295）显示女生均值 6.80（SD = 5.42），男生均值 6.15（SD = 5.19），效应量小（Cohen's d = 0.12）；住校生得分显著高于走读生（Wang et al., 2025）。普通成人社区研究中约 6–10% 的人得分≥15分，符合临床失眠标准。

重要局限：ISI 为主观自评量表，不能替代客观睡眠监测（多导睡眠监测 PSG）。量表不能区分原发性失眠与共病性失眠（如由抑郁、焦虑、睡眠呼吸暂停引起的失眠），若得分≥15分，强烈建议进行专业评估以排除其他睡眠障碍或躯体因素。本测评为研究教育用途，结果仅供参考，不构成任何诊断依据。
